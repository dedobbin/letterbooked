# letterbooked

Frontend package for `letterbooked`

## Requirements

-  pnpm (better package manager)
-  nvm (change Node versions easily)

## Setup

1. Install package dependencies:

   ```shell
   pnpm i
   ```

2. Run in dev mode:

   ```shell
   pnpm dev
   ```

3. Build everything:

   ```shell
   pnpm build
   ```
